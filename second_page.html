HTML

HTML — язык разметки для создания структуры веб-страницы и представления контента. Благодаря разметке браузер знает в каком порядке отображать элементы, и что они значат.

Основными тегами в HTML являются:

  • <!DOCTYPE>
  • <html>
  • <head>
  • <title>
  • <body>

CSS

CSS — язык каскадных стилей, который задаёт визуальное оформление для HTML, SVG и других языков разметки. Благодаря разметке браузер знает в каком порядке отображать элементы, и что они значат.

CSS-правило состоит из нескольких обязательных элементов:

  • Селектор
  • Свойство
  • Значение

JavaScript

JavaScript — язык программирования широкого спектра. На нём можно как создавать динамические интерфейсы, так и работать с базами данных и операционными системами.

В JavaScript используются такие типы данных как:

  • Примитивные
    1. Строка
    2. Число
    3. Булево
    4. Null
    5. BigInt
    6. Undefined
    7. Symbol
  • Структурные(комплексные)
    1. Объекты
    2. Массивы
